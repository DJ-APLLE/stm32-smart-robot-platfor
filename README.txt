一、项目搭建过程
1、硬件搭建
（一）底盘与驱动系统
安装麦克纳姆轮：将 4 个麦克纳姆轮安装在铝合金底盘轮轴上，确保轮子转动灵活且倾斜方向一致，用螺丝固定。
安装电机与驱动模块：固定 4 个直流电机并与轮轴对齐，2 个驱动模块各控制 2 个电机，按引脚定义连接电机正负极与控制线，固定驱动模块并确保散热良好。
（二）传感器与控制模块
超声波传感器与舵机：舵机固定于底盘前方或顶部，轴接传感器支架，信号线接控制板 PWM 引脚；超声波传感器 VCC 接电源，GND 接地，Trig 和 Echo 接控制板 IO 口。
OLED 显示屏：通过 IIC 接口（SDA、SCL）连接控制板，固定于底盘或支架以便观察。
（三）电源与交互模块
电源系统：安装电池盒（如 12V 锂电池），连接电源模块，输出端为控制板（5V）、驱动模块（12V）等供电，确保电压匹配。
语音识别模块：通过 UART 或 IO 口连接控制板，安装喇叭并固定。
（四）扩展模块（可选）
摄像头模块：固定 ESP32-CAM 于顶部，通过 WiFi 或串口与控制板通信。
AT24C02 存储芯片：通过 IIC 接口连接控制板，焊接或插座安装以存储安全距离。
2、软件编程
  核心功能实现
麦克纳姆轮运动：基于运动学原理编写全向移动函数，通过 PWM 和方向电平控制 4 个电机实现前进、后退、平移等动作。
超声波避障：读取传感器数据，当距离小于安全值时执行后退、转向等避障动作，安全距离可写入 AT24C02 存储。
模式与显示：实现手机 APP / 手动 / 自动模式切换，OLED 实时显示距离、速度、模式等参数。
语音与存储：集成语音模块实现指令识别，通过 IIC 协议读写 AT24C02 存储安全距离。
3、系统调试
（一）硬件调试
检查电路供电、电机转向与速度，校准超声波传感器测量精度。
（二）功能测试
测试手动 / 自动模式控制、避障、模式切换、语音识别等功能，优化安全距离和速度参数。



二、接线说明
麦克纳姆轮 A	PWM 	PE5 (TIM9_CH1)、    PE6(TIM9_CH2)
麦克纳姆轮 B	PWM	PC6(TIM8_CH1)、     PC7(TIM8_CH2)
麦克纳姆轮 C	PWM 	PC8(TIM8_C3)、       PC9(TIM8_CH4)
麦克纳姆轮 D	PWM 	PB6(TIM4_CH1)、     PB7(TIM4_CH2)
蓝牙模块	        USART3	PB10（TX）、          PB11（RX）
超声波模块	        USART2	PA2（TRIG）、        PA3（ECHO）
语音控制模块	USART1	PA9（TX）、            PA10（RX）
摄像头模块	        DCMI 接口	按 DCMI 标准引脚分配（见数据手册）
舵机模块	        PB0(PWM信号线)	               PB0  (TIM3_CH3)



三、数据协议
1.手机-蓝牙模块：USART通信
2.控制板-超声波模块：UART通信
3.控制板-电机：PWM
4.控制版-OLED：硬件IIC同信
5.控制板-语音ASR01模块UART通信
6.控制板-摄像头：WIFi通过HTTP协议传输
